Questo programma permette di accedere velocememente alle stanze Webex di tre professori


PER USARE CORRETTAMENTE IL PROGRAMMA:


1)Recatevi nella cartella di installazione e modificate il file config.bat (ad esempio usando apri con Blocco Note) per impostare i link alle stanze
Webex e i nomi dei professori. Potete anche modificare il browser con cui lanciare il programma, che verrà chiuso al termine dell'operazione
quindi mettetene uno che non rischiate di avere già aperto e di perdere anche le altre schede.

2)Scaricate e Installate Cisco Webex Meetings da https://www.webex.com/downloads.html

3)Eseguite l'accesso a Cisco Webex Meetings con le credenziali del Poli

4)Aprite il browser selezionato al passaggio 1 (se lo avete saltato è Microsoft Edge), recatevi su https://www.webex.com/it/index.html
e accedete con le credenziali del Poli

5)Spostate il file Webex Launcher.ink sul desktop

5)Enjoy!
